# Backend feito em springboot

## Dependencias

1. Spring Web: Necessidade para criar backend para aplicações web
2. Spring Boot DevTools: Para ferramentas de auxilio ao desenvolvimento
3. Lombok : Para reduzir o codigo em classe, reduzindo a necessidade de Getters, Setters e construtores
4. Spring Security: Para proteção contra XSS, CSRF entre outros
5. Spring Data JPA: Criação de entidades e poder usar o banco a partir de um modelo ORM
6. MySQL Driver: Para conexão a banco mysql
7. Validation: Validação
8. Java Mail Sender: Envio e costumização de emails
9. MapperStruct: Conversão automatica de DTOs para Entities
